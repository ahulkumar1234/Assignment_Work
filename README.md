# 🚀 Franchise Landing Page

A modern, responsive landing page designed for promoting franchise opportunities. This project showcases a clean UI, smooth user experience, and professional design suitable for real-world business use.

---

## 🌐 Live Demo

👉 https://assignment-work-mu.vercel.app

---

## 📌 Features

* ✅ Modern & Clean UI (Dark + Blue Theme)
* ✅ Fully Responsive Design
* ✅ Smooth Hover Effects & Animations
* ✅ Professional Card Layout
* ✅ Glassmorphism Contact Form
* ✅ Scroll Animations (AOS Library)
* ✅ Call-To-Action Sections

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Transitions, Glassmorphism)
* JavaScript (Basic Form Handling)
* AOS (Animate On Scroll Library)

---

## 📂 Project Structure

```
project/
│── index.html
```

## 🎯 Objective

The goal of this project is to create a professional landing page for a franchise business that clearly communicates:

* Business overview
* Benefits of franchising
* Comparison with startups
* Lead generation through contact form

---

## 📸 Sections Included

* Hero Section
* Business Overview
* Franchise Benefits
* Startup vs Franchise Comparison
* Contact / Inquiry Form
* Footer
